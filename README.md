# 🎧 Mood-Based Music Player (Face Detection)

A smart web application that detects user facial expressions in real-time and plays music based on the detected mood.

## 🚀 Features

- 🎭 Real-time face detection using MediaPipe
- 😊 Detects emotions like Happy, Sad, Neutral, Surprise
- 🎵 Automatically plays songs based on mood
- 🔄 Dynamic frontend-backend integration
- 🌐 Full-stack project (React + Node.js + Express)

## 🛠️ Tech Stack

**Frontend:**
- React.js
- SCSS

**Backend:**
- Node.js
- Express.js

**AI / ML:**
- MediaPipe (Face Detection)

**Others:**
- ImageKit (for media storage)
- Node-ID3 (for song metadata)
